The Mission

RoboSoccer 4.0 is an online robotics coding challenge built around a custom CoppeliaSim simulation. Participants receive a ready-made scene that includes the robot, ball, obstacles, and goal area.

Event Abstract

"The task is to write autonomous logic inside the simulator to move the bot and score goals. We provide clear documentation for installing and setting up CoppeliaSim on both Windows and Linux to ensure a smooth start for all, even simulation beginners."

Challenge Timeline

01

Setup & Scripting

Download the scene file and installation guides. Participants get 7-8 days to implement autonomous logic.

02

Round 1 Submission

Submit your scene file and a demonstration video showing the robot successfully scoring goals.

03

Live Finals

A 3-minute real-time showdown on Google Meet. Most goals wins. Tie-breakers conducted if necessary.
